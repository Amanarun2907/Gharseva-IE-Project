## How to Add Images:

1. Save each image from your collection with the corresponding name above
2. Place them in this `frontend/public/images/` directory
3. The images will automatically appear on the website
4. If an image is missing, the component will gracefully hide it (no broken image icons)

## Image Guidelines:

- Format: JPG (as provided)
- Recommended size: 800x600px or similar aspect ratio
- Keep file sizes under 500KB for faster loading
- Use descriptive alt text for accessibility
